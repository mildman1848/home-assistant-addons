# Home Assistant Add-on: Neolink

Neolink bridges Reolink cameras that only expose the proprietary Reolink/Baichuan protocol, commonly on TCP port `9000`, to standard RTSP.

Typical stream URLs after installation:

```text
rtsp://HOME_ASSISTANT_IP:8554/reolink_e1/mainStream
rtsp://HOME_ASSISTANT_IP:8554/reolink_e1/subStream
```

## Install

Add this repository to Home Assistant:

```text
https://github.com/mildman1848/home-assistant-addons
```

Then install **Neolink** from the add-on store.
